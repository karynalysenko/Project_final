Usage of PhaTYP

After installation of the tool, to use in a local machine, steps must be taken:
 -> activate phabox environment, go to Phabox folder installation, open terminal, and type "conda activate phabox"
 -> phatyp.py file must be run on conda environment

This script will automatically run through all genome folders of "data", and save correspondent PhaTYP result in a new folder of same accession_id in phatyp_results folder.

The folder Phatyp already contains the results of this project, named phatyp_results.zip. To use these results, user must to unzip the folder first.
